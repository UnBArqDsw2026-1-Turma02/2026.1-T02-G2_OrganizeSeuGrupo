# OrganizeSeuGrupo

## Sobre

Este repositório é destinado ao desenvolvimento da primeira entrega do projeto do grupo 02 da disciplina de Arquitetura e Desenho de Software, ministrada pela professora Milene Serrano.

## Entrega 01

A primeira entrega aborda todo o escopo requisitado, considerando os conceitos e fases do Design Sprint, a elaboração de artefatos generalistas e, por fim, a modelagem BPMN. Todas divididas em diferentes tópicos de entrega pelo GitPages:

#### 1. Desenho de Software (Base)

1.1 Design Sprint

1.2 Artefatos Generalistas

1.3 Modelagem BPMN

## 📚 Documentação

A geração da documentação no Github Pages é realizada utilizando o [docsify](https://docsify.js.org/).

```shell
"Docsify generates your documentation website on the fly. Unlike GitBook, it does not generate static html files. Instead, it smartly loads and parses your Markdown files and displays them as a website. To start using it, all you need to do is create an index.html and deploy it on GitHub Pages."
```

### Instalando o docsify

Execute o comando:

```shell
npm i docsify-cli -g
```

### Executando localmente

Para iniciar o site localmente, utilize o comando:

```shell
docsify serve ./docs
```
