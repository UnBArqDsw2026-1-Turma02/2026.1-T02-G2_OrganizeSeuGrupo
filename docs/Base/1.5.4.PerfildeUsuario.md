# Perfil de Usuário

## Introdução

Este documento tem como objetivo definir um perfil de usuário com base nas respostas do [Questionário](./Base/1.5.3.Questionario.md).

## Metodologia

A coleta de dados foi estruturada por meio de um formulário digital via Google Forms, composto por questões objetivas e subjetivas para viabilizar análises quantitativas (frequência de uso) e qualitativas (principais dificuldades na organização de grupos). O questionário foi aplicado entre os dias 03/04 e 05/04, distribuído em grupos. Essa conversa direta com o pessoal da faculdade trouxe informações reais sobre como eles se organizam, o que foi essencial para decidirmos o que o OrganizeSeuGrupo precisa ter para ser útil de verdade.

## Perfil de Usuário

O perfil de usuário foi elaborado a partir de uma análise detalhada dos resultados do questionário aplicado a estudantes. Utilizamos dados demográficos, comportamentais e funcionais para identificar os padrões e as nuances que definem a experiência de organizar e participar de grupos de estudos. Esta análise considerou aspectos como atitudes frente ao trabalho coletivo, expectativas sobre ferramentas de apoio e as tarefas primárias realizadas no cotidiano acadêmico.

<center>

**Tabela 1: Perfil de Usuário**

| Características | Descrição |
| :--- | :--- |
| **Idade:** | Predominantemente 18-24 anos. |
| **Gênero:** | Masculino.  |
| **Experiência com Tecnologia:** | Alta. Estudantes habituados a ferramentas digitais, redes sociais e aplicativos de produtividade. |
| **Escolaridade:** | Ensino Superior Incompleto (estudantes de graduação). |
| **Frequência em Grupos de Estudo:** | Regular, porém com dificuldades de consistência devido à gestão de tempo. |
| **Canal de informação/Notificação:** | Preferência por notificações no aplicativo. |
| **Atitudes:** | Busca por praticidade e agilidade; valoriza a colaboração, mas sente-se frustrado com a falta de responsabilidade de membros. |
| **Expectativas:** | Encontrar parceiros com interesses similares e ter ferramentas que facilitem o cumprimento de cronogramas. |
| **Tarefas Primárias:** | Buscar grupos por disciplina; conciliar horários de reunião; compartilhar arquivos/resumos e gerir listas de tarefas. |
