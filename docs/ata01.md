## 1. Pauta da Reunião

## 2. Tópicos Discutidos

### 2.1 Brainstorming e Metodologia
*   O grupo realizou uma sessão de ideias abertas para o direcionamento do app **OrganizeSeuGrupo**.
*   **Gabriel** disponibilizou um quadro no **Miro**, onde todos os integrantes puderam registrar suas sugestões e contribuições visualmente.
*   **Mayara** apresentou a metodologia de levantamento de requisitos utilizada por ela no ano anterior na matéria de requisitos, servindo como base para estruturar o fluxo de trabalho.
*   **Todos** discutiram sobre a possibilidade de realizar um Rich Picture para concepção inicial do problema.
*   **Eduardo** e **Luiza** estruturaram a parte inicial do Miro, utilizando o material disponibilizado em aula como base.

### 2.2 Definição do Problema e Objetivo
*   Foi realizada uma análise baseada em um **Case Study** para alinhar a compreensão do grupo sobre o problema real a ser enfrentado.
*   O objetivo principal do projeto foi validado com base nessa análise.

### 2.3 Próximos Passos (Design Sprint - Dia 3)

*   Definição de primeiros passos para a prototipagem, além de definir linhas de estratégia para o dia 5 -- Test.
*   Definição da separação dos integrantes em conjuntos para a execução dos artefatos similares.
